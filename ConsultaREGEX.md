# Regex

Material criado para consulta rápida das regex que criei e utilizo com frequência.

As regex foram utilizadas, principalmente, em conjunto com as linguagens JavaScript e Java.

Javascript:

<code>\gi</code>

Java:

<code>\gms</code>

1. **Captura de CPF**


<code>\d{3}\.\d{3}\.\d{3}-\d{2}<\code>
<code>\d{3}\.?\d{3}\.?\d{3}-?\d{2}<\code>
